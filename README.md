# Re_ADOJAS
A multi-platform light ADOFAI Level Player.

## Stacks
Vite

React

THREE.JS

[ADOFAI-JS](https://github.com/adofaiex/ADOFAI-JS)

## Support
WEBGL/WEBGPU

## Deploy
```bash
git clone https://github.com/Maicy0609/ADOR3T.git ADOR3T
cd ADOR3T
pnpm install
# npm install
# yarn install
pnpm dev
```
Then launch your browser and visit `http://localhost:5173` for development.
> If viewing in LAN is required,use `pnpm dev --host 0.0.0.0` instead.
